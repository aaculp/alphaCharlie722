## üåü Quick Tips

1. **Enable notifications on first launch** - Don't skip the permission prompt
2. **Customize your preferences** - You don't have to receive all notification types
3. **Check device settings** - App settings and device settings must both be enabled
4. **Preferences sync automatically** - Set once, applies to all devices
5. **In-app notifications always work** - Even if push is disabled
6. **Battery efficient** - Push notifications use minimal battery
7. **Works when app is closed** - Never miss an update
8. **Tap to navigate** - Notifications take you directly to the relevant screen

## üìñ Glossary

**Push Notification:** Alert sent to your device via Firebase Cloud Messaging  
**In-App Notification:** Notification stored in the app's notification center  
**Device Token:** Unique identifier for your device used to send notifications  
**FCM:** Firebase Cloud Messaging, the service that delivers push notifications  
**Notification Preferences:** Your settings for which notifications you receive  
**Master Toggle:** Main switch that controls all push notifications  
**Notification Tray:** System area where notifications appear on your device  
**Foreground:** When the app is open and active  
**Background:** When the app is running but not visible
