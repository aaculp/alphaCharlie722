## Progressive Disclosure

### Level 1: New User
- 4 core cards (all showing 0)
- Clean, simple interface
- Encourages first check-in

### Level 2: First Activity
- Core cards show real numbers
- First conditional card appears
- User sees progress

### Level 3: Regular User
- Multiple conditional cards
- Profile feels personalized
- Gamification kicks in

### Level 4: Power User
- All cards visible
- Rich data visualization
- Strong engagement
